# 开篇引言：为什么用 Zig 开发单片机？

## 所属模块
模块一：入门基础篇

## 学习目标
- 了解 Zig 语言的核心特性
- 理解 Zig 在单片机开发中的优势
- 掌握本教程的学习路线
- 了解硬件平台选型

## 内容要点
- Zig 语言核心特性（无GC、内存安全、编译期计算、兼容C）
- Zig 与 C/Rust 在单片机开发中的优势对比
- 本教程学习路线与目标说明
- 硬件平台选型推荐（星火1号，主芯片STM32F407）及采购建议

## 硬件需求
- 星火1号开发板（STM32F407）
- USB 数据线

## 软件需求
- 无需特殊软件，本节为理论介绍

## 学习时长
约 15 分钟

## 常见问题
Q: 为什么选择 Zig 而不是 C 或 Rust？
A: Zig 结合了 C 的简洁性和 Rust 的安全性，更适合嵌入式开发。

Q: 必须使用星火1号开发板吗？
A: 不是必须的，但本教程基于该平台设计，使用其他平台需要自行适配。
