# Zig 单片机从0到1实战教程

> 面向零基础/基础薄弱的嵌入式爱好者，打造"从环境搭建到实战项目"的全流程 Zig 单片机入门教程

## 教程介绍

本教程系列旨在帮助嵌入式爱好者从零开始掌握 Zig 语言进行单片机开发。通过系统的学习路径，从基础语法到外设驱动，再到完整实战项目，让你全面掌握 Zig 在嵌入式领域的应用。

### 硬件平台

**推荐开发板：星火1号（STM32F407）**

- 主芯片：STM32F407
- 成本亲民，资料完备
- 硬件通用，扩展方便
- 适合新手入门

### 目标受众

- C/嵌入式开发者：想尝试 Zig 语言，了解其在单片机领域优势的现有开发者
- 技术探索者：关注新兴语言在嵌入式领域应用的技术爱好者

1. 准备硬件：星火1号开发板（STM32F407）
