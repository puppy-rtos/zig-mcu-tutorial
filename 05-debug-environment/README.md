## 学习目标

掌握在 VSCode 中搭建 Zig MCU 单步调试环境的方法：
- 安装必要的调试工具
- 配置 VSCode 调试环境
- 使用 ST-Link 进行硬件调试
- 掌握 pyocd 调试命令
- 实现 Zig 代码的单步调试

## 示例代码

示例项目包含以下文件：

- `src/main.zig` - 调试示例代码
- `src/link.ld` - 链接脚本
- `.vscode/launch.json` - VSCode 调试配置
